# Codyssey_A1_1 Project : A1_1_Python 콘솔 프로그램

* 프로그램 이름과 간단한 설명
  - A1_1_Project.py
  - 자산운용관련 보고서 작성 프롬프트 관리 프로그램

* 실행 방법
  - 메인메뉴에서 해당하는 번호를 입력하면 해당 기능을 실행
  - 1번 메뉴 : 프롬프트 추가
  - 2번 메뉴 : 프롬프트 목록 보기
  - 3번 메뉴 : 카테고리별 조회
  - 4번 메뉴 : 프롬프트 검색
  - 5번 메뉴 : 상세 보기 및 즐겨찾기 설정
  - 6번 메뉴 : 프로그램 종료
    
* 기능 목록
  - add_prompt() : 프롬프트 추가 모듈
  - view_list() : 프롬프트 목록 보기 모듈
  - view_by_category() : 카테고리별 조회 모듈
  - search_prompt() : 프롬프트 검색
  - view_details() : 상세 보기 및 즐겨찾기 설정 모듈
 
* 등록된 프롬프트 카테고리 : 자산운용 큰 카테코리
  - 주식, 채권, 디지털자산, 부동산, 원자재
